# 3D Data Science & BI Portfolio

An immersive, interactive 3D portfolio website built for a Data Scientist and BI Developer. This project showcases professional experience, technical skills, and projects using stunning 3D graphics and smooth animations.

## 🌟 Features

*   **Interactive 3D Hero Section:** Features a dynamic 3D computer model using Three.js and React Three Fiber.
*   **Smooth Animations:** Powered by Framer Motion for seamless page transitions and scroll reveals.
*   **Infinite Skills Marquee:** A continuously scrolling display of technical skills and tools.
*   **Interactive Project Cards:** 3D tilt effects on project cards using `react-parallax-tilt`.
*   **Experience Timeline:** A vertical timeline detailing professional work history.
*   **Functional Contact Form:** Integrated with FormSubmit for direct messaging without a backend.
*   **Fully Responsive:** Optimized for both desktop and mobile devices.

## 🛠️ Tech Stack

*   **Framework:** [React 19](https://react.dev/)
*   **Build Tool:** [Vite](https://vitejs.dev/)
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/)
*   **3D Graphics:** [Three.js](https://threejs.org/) & [@react-three/fiber](https://docs.pmnd.rs/react-three-fiber/)
*   **Animations:** [Framer Motion](https://motion.dev/)
*   **Icons:** [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/dhiraj-jadhav-portfolio.git
   cd dhiraj-jadhav-portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:3000`

### Building for Production

To create a production-ready build, run:

```bash
npm run build
```

The compiled assets will be available in the `dist` directory, ready to be deployed to platforms like Vercel, Netlify, or GitHub Pages.
